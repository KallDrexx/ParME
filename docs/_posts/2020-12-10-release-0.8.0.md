Parme version `0.8.0` has been released.  

## Changes

* Editor fonts are much clearer
* Images displayed in the editor now adhere to the sampling selected in the view menu
* The texture section editor now allows zooming and panning of the sprite sheet.  It also can provide grid lines to help users figure out the pixel values when creating new texture sections.
* When a texture section is selected that area is now highlighted on the sprite sheet.
* The editor now displays recently opened files in the File menu
* Velocity based rotation no longer changes the rotation of particles with zero velocity
* FlatRedBall's `ParmeEmitterManager` now cleans itself up on screen changes.
